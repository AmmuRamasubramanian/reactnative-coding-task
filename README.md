# 🚗 Mileage Tracker Lite App

A cross-platform mobile application built with **React Native** that helps users track vehicle fuel usage, monitor consumption, and manage refueling records efficiently.

---

## ✨ Features

- 🚗 Add & manage multiple vehicles  
- ⛽ Log refueling records  
- 📊 Track fuel consumption (average & latest)  
- 📅 Sort records by date  
- ⚡ Optimized performance using memoization  
- 🧠 Smart validation for odometer readings  
- 📱 Clean and responsive UI  

---

## 🛠️ Tech Stack

- **React Native**
- **Zustand** (State Management)
- **JavaScript (ES6+)**
- **FlashList** (for optimized list rendering)
